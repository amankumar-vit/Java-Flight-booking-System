# ✈️ Flight Booking System

A simple console-based Flight Booking System developed in Java. This project allows users to book flights, select seats, make payments, and receive booking confirmation through a user-friendly command-line interface.

## 🚀 Features

- View available flights
- Select a flight route
- Choose seat type (Window, Middle, or Aisle)
- Enter passenger details
- Payment verification
- Booking confirmation generation
- Input validation

## 🛠️ Technologies Used

- Java
- Arrays
- Switch Case
- Conditional Statements
- Scanner Class

## 🎯 Learning Objectives

This project helps in understanding:

- Java Fundamentals
- User Input Handling
- Arrays
- Decision Making Statements
- Console-Based Application Development
